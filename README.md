# FancyTodo-Server
>Hello! Welcome to Fancy Todo App!<br>
>Below are the list of URL that you can use<br>

| URL               | Method        |
| -------------     |:-------------:|
| /users/register   | POST          |
| /users/login      | POST          |
| /todos            | GET           |
| /todos            | POST          |
| /todos/:id        | GET           |
| /todos/:id        | PUT           |
| /todos/:id        | DELETE        |
| /public-holidays  | GET           |
<br><br>

## /users/register
-----
* method: POST
* purpose: Register account for new user
* req.body: <br>
    ```javascript
        {
            "name": "tina",
            "email": "tina@contoh.com",
            "password": "abcdef",
        }
    ```
* success response: <br>
    * code: 200 <br>
    * content: <br>
    ```javascript
        {
            "User": {
                "id": 5,
                "name": "tina",
                "email": "tina@contoh.com",
                "password": "$2a$10$vGDNHEv6eVd5t1m1z6Tske31y69h7xs346S7iwiKwwIo9d3T2vnDm",
                "updatedAt": "2020-04-28T07:21:21.651Z",
                "createdAt": "2020-04-28T07:21:21.651Z"
            }
        }
    ```
* error response: <br>
    * code: 400 <br>
    * content: <br>
    ```javascript
    {
        "msg": "please input data as required"
    }
    ```

    OR
    * code: 400 <br>
    * content: <br>
    ```javascript
    {
        "msg": "please make sure your email never registered before"
    }
    ```

    OR
    * code: 500 <br>
    * content: <br>
    ```javascript
    {
        "msg": "internal server error"
    }
    ```
